## 概要
AWSのARNを基に正規表現を使用してアカウントIDを抽出します。
ググっても見つからない＆なんだかんだ使う場面が多いのでメモ

## スクリプト
```shell
echo <ARN> | sed -e 's/^.*::\([0-9]*\).*$/\1/'
```

## 実行結果
```shell
$ echo "arn:aws:iam::123456789012:role/one-liner-role" | sed -e 's/^.*::\([0-9]*\).*$/\1/'
123456789012
```
